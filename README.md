# LibrifyJS: libgen.me

A Firefox extension for replacing the JavaScript blocked by GNU LibreJS on libgen.me
with free / libre code.

## Status

### What Works

- Search form (press enter to submit) and result display
- Item details
- Item link list

### What Doesn't Work

- Probably anything else
